Assignment 2: City Management

This module provides utility functions to manage a list of city names.

Functions implemented in list_utils.py:
1. get_unique_cities(cities) - Returns a new list of unique city names, preserving order.
2. get_sorted_cities(cities, descending=False) - Returns a sorted copy of the cities list.
3. convert_cities_to_upper(cities) - Converts all city names to uppercase using list comprehension.

Testing:
- test_list_utils.py imports the functions.
- Creates a sample city list with duplicates.
- Calls each function and prints results.
- Confirms that the original list remains unchanged.

Learning Objectives:
- Removing duplicates without using set.
- Sorting lists with optional descending order.
- Using list comprehension for transformations.
- Practicing modular programming.
