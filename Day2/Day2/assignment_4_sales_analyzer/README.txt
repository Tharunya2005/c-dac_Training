Assignment 4: Sales Analyzer

This module provides utility functions to analyze sales data.

Functions implemented in list_utils.py:
1. filter_sales_above(sales, threshold) - Returns sales values above a threshold using list comprehension.
2. get_sales_summary(sales, first_n, last_n) - Returns first and last n sales using slicing.
3. get_reversed_sales(sales) - Returns a reversed copy of the sales list.
4. get_sorted_sales(sales, descending=False) - Returns a sorted copy of the sales list.

Testing:
- test_list_utils.py imports the functions.
- Creates a sample sales list.
- Calls each function and prints results.
- Verifies that the original list remains unchanged.

Learning Objectives:
- Filtering with list comprehension.
- Using slicing for summaries.
- Reversing and sorting lists safely.
- Practicing modular programming and testing.
